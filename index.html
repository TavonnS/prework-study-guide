HTML

  • HTML creates the structure of a webpage, using <> tags.
  • It's an acronym for hypertext markup language

CSS

  • Cascading Style Sheets provide the decoration, theme, and aesthetic design of a webpage.
  • CSS can code color, size, margin, layout, and other visual formatting.

Git

  • Git manages the development of webpages with version control and collaboration.
  • Git uses a branching method to keep track and organize a web project.
  • It is useful when improving or debugging code, so as not to lose earlier versions of the project.

JavaScript

  • JavaScript makes webpages interactive and capable of logical tasks.
  • JavaScript uses elements, such as functions, arrays, Booleans, and loops, which are written into the JS section of a webpage.
  • These JS elements allow a user to interact in certain ways with website contents.
